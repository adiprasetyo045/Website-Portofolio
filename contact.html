Apakah Anda tersedia untuk pekerjaan freelance atau magang?

Ya, saya saat ini terbuka untuk proyek freelance dan peluang magang, terutama dalam teknologi kesehatan, pembelajaran mesin, dan pengembangan full-stack. Jangan ragu untuk menghubungi saya dengan detail tentang proyek atau peluang Anda.

Apa stack teknologi yang Anda gunakan untuk proyek pengembangan web?

Untuk pengembangan frontend, saya lebih suka menggunakan HTML5, CSS3, dan vanilla JavaScript atau React.js. Untuk backend, saya biasanya menggunakan Python dengan Flask atau Django, atau Node.js dengan Express. Untuk database, saya memiliki pengalaman dengan MySQL, PostgreSQL, dan MongoDB.
